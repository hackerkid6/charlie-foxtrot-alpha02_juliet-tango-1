QM3

Model Misspecification

▶

The One Rule That Covers All Three Violations

Heteroskedasticity, serial correlation, and multicollinearity all bias standard errors but NOT the regression coefficients themselves. Coefficients remain unbiased — only inference (t-stats, F-stats) breaks down.

ViolationBiases β̂?Biases SE?Effect on t-statsDetection TestFix
Conditional HeteroskedasticityNoYesUnreliableBreusch-PaganWhite's robust SE
Positive Serial CorrelationNoYes (understated)Too high → Type I ↑Durbin-WatsonNewey-West / Hansen
MulticollinearityNoYes (overstated)Too low → Type II ↑VIFDrop a variable

Heteroskedasticity

Breusch-Pagan Test
Regress ε̂²ᵢ on all X's → get R² from that auxiliary regression
BP = n × R² ~ χ²(k)
If BP > χ²_critical → conditional heteroskedasticity exists
Example
n=100, k=3, R² from aux regression = 0.09
BP = 100 × 0.09 = 9.0 vs χ²(3) at 5% = 7.81
9.0 > 7.81 → Reject H₀ → heteroskedasticity present
Fix: report White's robust standard errors. Coefficients stay the same.

Serial Correlation

Durbin-Watson
DW = 2(1 − r̂) where r̂ = first-order autocorrelation of residuals
DW ≈ 2 → clean | DW → 0 → strong positive SC | DW → 4 → strong negative SC
Compare to d_L, d_U: DW < d_L → reject H₀ (SC exists)
Positive serial correlation UNDERSTATES standard errors → t-stats look significant when they're not (Type I error inflation). This is the #1 tested consequence.
In AR models, DON'T use DW — use residual autocorrelation tests. DW is invalid when lagged Y is a regressor.

Multicollinearity

Classic sign: high R² and significant F-stat, but individually insignificant t-stats.

VIF
VIF_j = 1 / (1 − R²_j) where R²_j = R² from regressing X_j on all other X's
VIF < 5: OK | 5–10: moderate | > 10: severe
Example
Regress X₁ on X₂, X₃ → R²₁ = 0.90
VIF₁ = 1/(1-0.90) = 10 → severe multicollinearity
Coefficients are still correct on average — but SEs so inflated that nothing is statistically significant. Fix: drop X₁ or X₂ (whichever is less theoretically important).

Regression Extensions: Influential Obs, Dummies, Logistic

▶

Influential Observations

TermDefinitionDetectionThreshold
High-leverage pointExtreme X valueLeverage h_iih_ii > 3(k+1)/n
OutlierExtreme Y (large residual)Studentized residual t*|t*| > 3
Influential observationBoth extreme X and Y → tilts regression lineCook's DCook's D > 1
High leverage alone ≠ influential. Outlier alone ≠ influential. Cook's D captures the combined effect — whether the observation actually PULLS the regression line toward itself.

Dummy Variables

The dummy variable trap: for n categories, use n−1 dummies. Including all n creates perfect multicollinearity — regression fails.
Intercept Dummy (parallel shift)
Y = b₀ + d₀D + b₁X + ε
D=0: Y = b₀ + b₁X (base group) | D=1: Y = (b₀+d₀) + b₁X
Slope Dummy (interaction — steepness changes)
Y = b₀ + b₁X + d₁(D·X) + ε
D=0: slope = b₁ | D=1: slope = b₁ + d₁

Logistic Regression

Logistic Regression (binary Y)
ln[p/(1−p)] = b₀ + b₁X₁ + ... + bₖXₖ
Probability: p = 1 / (1 + e^{−(b₀+b₁X₁+...)}
Odds ratio = e^(bⱼ) = multiplicative change in odds per unit ↑ in Xⱼ
Estimated via MLE, not OLS. Significance tested with z-test or likelihood ratio test.
Interpreting Logistic Coefficients
Coefficient on debt-to-equity = 1.2
Each 1-unit ↑ in D/E → log-odds of default ↑ 1.2
Odds ratio = e^1.2 = 3.32 → odds of default multiply by 3.3x per unit ↑ in D/E
Positive coefficient → increases probability of Y=1. Negative → decreases it.
QM5

Time Series: AR Models, Unit Roots, ARCH

▶

Trend Models

Linear vs Log-Linear
Linear: y_t = b₀ + b₁t + ε_t → grows by constant AMOUNT per period
Log-linear: ln(y_t) = b₀ + b₁t + ε_t → grows by constant RATE (exponential)
If linear trend residuals show serial correlation → switch to log-linear or AR model

AR Models & Stationarity

AR(1) Model
x_t = b₀ + b₁x_{t-1} + ε_t
Covariance-stationary requires: |b₁| < 1
Mean-reverting level = b₀ / (1 − b₁)
Mean Reversion Example
b₀ = 0.4, b₁ = 0.8 → mean-reverting level = 0.4 / 0.2 = 2.0
If x_t = 5 (above mean): x_{t+1} = 0.4 + 0.8(5) = 4.4 → moves toward 2.0
If x_t = 0 (below mean): x_{t+1} = 0.4 + 0.8(0) = 0.4 → moves toward 2.0

Unit Root & Dickey-Fuller Test

Dickey-Fuller Test
Random walk: x_t = x_{t-1} + ε_t (b₁ = 1, nonstationary, no mean reversion)

DF reformulation: Δx_t = γ·x_{t-1} + ε_t where γ = b₁ − 1
H₀: γ = 0 (unit root) | H₁: γ < 0 (stationary)
Reject H₀ if DF test stat is MORE NEGATIVE than DF critical value
If unit root → first-difference the series: use Δx_t instead
DF critical values are MORE NEGATIVE than standard t-critical values. At 5%, DF ≈ −2.86 vs t ≈ −1.65. Using t-tables here = wrong conclusion.
Cointegration: two nonstationary series where a linear combination IS stationary. Run regression, then DF test on residuals. If residuals are stationary → cointegrated → regression is valid.

ARCH

ARCH(1) — Volatility Clustering
Conditional variance: σ²_t = a₀ + a₁ε²_{t-1}
Significant a₁ → large shocks followed by large variance
Detect: regress ε̂²_t on ε̂²_{t-1} → if slope significant → ARCH | Fix: GARCH models

Bias-Variance Tradeoff

Total Error Decomposition
Total Error = Bias² + Variance + Irreducible Noise
High bias = underfitting (too simple) | High variance = overfitting (too complex)
Goal: minimize both via cross-validation, regularization, simpler model
Intuition
Straight line through 20 points: high bias (misses the real curve), low variance.
20th-degree polynomial: perfect on training data, terrible on new data = high variance, low bias.
A 3rd-degree polynomial balances both → best out-of-sample performance.

Classification Metrics

Key Metrics
Accuracy = (TP+TN) / Total
Precision = TP / (TP+FP) ← when you predict positive, how often right?
Recall = TP / (TP+FN) ← of all actual positives, how many caught?
F1 = 2 × Precision × Recall / (Precision + Recall)
AUC-ROC: higher → better discrimination between classes

Currency Exchange Rates & Parity Conditions

▶

FX Conventions

Quote = P/B (Price / Base). USD/EUR = 1.35 → 1 EUR costs 1.35 USD. "Base in the base." Bid = dealer buys base (lower). Ask = dealer sells base (higher).

Cross Rate
A/C = (A/B) × (B/C) — B cancels
Bid of cross = bid₁ × bid₂ | Ask of cross = ask₁ × ask₂
Inverting a quote: new bid = 1/old ask | new ask = 1/old bid

Triangular Arbitrage

DBi G IO (dealer bid > interbank offer): buy from interbank, sell to dealer.
DO L IB (dealer offer < interbank bid): buy from dealer, sell to interbank.

Number line trick: dealer quote entirely LEFT of interbank range → DO L IB (buy from dealer). Entirely RIGHT → DBi G IO (buy from interbank). Overlapping = no arb.

Forward Rate & Covered Interest Rate Parity (CIP)

Forward Rate Formula
F_{P/B} = S_{P/B} × [1 + i_P(d/360)] / [1 + i_B(d/360)]
If i_P > i_B → F > S (base currency at forward discount)
If i_P < i_B → F < S (base currency at forward premium)
Example
Spot CAD/AUD = 1.0145, 270-day AUD rate = 4.87%, CAD rate = 1.41%
F = 1.0145 × [1 + 0.0141(270/360)] / [1 + 0.0487(270/360)] = 0.9891
AUD is at a forward discount because its interest rate is higher than CAD's.

All Five Parity Conditions

ConditionFormulaHolds?

UIP (Uncovered IRP)%ΔS_expected = i_P − i_BFails short-term → carry trade profits
PPP (relative)%ΔS ≈ π_P − π_BLong-run only

Carry Trade Intuition
UIP says: borrow JPY at 0.5%, invest in AUD at 5% → net 0% after AUD depreciates.
In practice, AUD often doesn't depreciate enough → carry trade pockets the rate differential.
Risk: in risk-off environments, carry trades unwind violently (AUD crashes fast).

Economic Growth & Investment Implications

▶

Why Growth Matters

Stock Price Identity
P = GDP × (E/GDP) × (P/E)
Long-run: E/GDP and P/E mean-revert → stock prices track GDP growth
Long-run real EPS growth cannot persistently exceed potential GDP growth

Production Function & Growth Accounting

Cobb-Douglas
Y = TFP × K^α × L^(1−α) [α ≈ 1/3]
ΔY/Y = ΔTFP/TFP + α(ΔK/K) + (1−α)(ΔL/L)
TFP = Solow residual = technology-driven growth (what's left after capital and labor)

Growth Models

ModelKey ClaimLong-Run Growth DriverConvergence?
ClassicalTechnology → population ↑ → income falls to subsistence (Malthusian)None sustainablyAll → subsistence wage
Neoclassical (Solow)Diminishing returns to capital → steady state; only TFP drives long-run per capita growthExogenous TFPConditional (same parameters → converge)
EndogenousNo diminishing returns to broad capital (human capital, R&D); knowledge spills overPolicy, R&D, human capitalNo necessary convergence
Neoclassical Steady State — The Key Intuition
Adding capital per worker boosts growth temporarily, but with diminishing returns, each additional machine adds less output than the last.
Eventually, new investment just replaces depreciation → no additional growth.
Only sustained TFP growth (better technology, processes, institutions) permanently raises the per capita growth rate.

Intercorporate Investments

▶

Classification Framework

OwnershipInfluenceAccounting Method
<20%NoneFinancial assets: FVPL, FVOCI, or Amortized Cost
20–50%SignificantEquity Method
Shared controlJointEquity Method (IFRS)
>50%ControlFull Consolidation
The 20%/50% thresholds are guidelines. Classification depends on DEMONSTRATED influence — a 15% stake with board representation can still be classified as an associate.

Equity Method Mechanics

Investment Balance
Ending balance = Beginning balance + Share of net income − Share of dividends received
(Dividends received = return of capital, NOT income on the income statement)
Worked Example
Branch pays €200,000 for 20% of Williams. Over 3 years: Williams earns €900,000 total, pays €350,000 dividends.
Investment balance = €200,000 + 20%(€900,000) − 20%(€350,000)
= €200,000 + €180,000 − €70,000 = €310,000
Income statement: Branch reports €180,000 equity income (20% of €900,000).
The €70,000 in dividends reduces the investment, not income.

Excess Purchase Price & Goodwill

Decomposing an Acquisition Price
Pay $40M for 25% of a company with BV net assets $100M, FV net assets $120M.
25% × BV($100M) = $25M → pure book value interest
25% × ($120M − $100M) = $5M → excess FV over BV (amortized over asset lives, reduces equity income)
Goodwill = $40M − $25M − $5M = $10M → not amortized, tested annually for impairment

Acquisition Method (Consolidation)

  • Record 100% of assets/liabilities at FAIR VALUE on acquisition date
  • Goodwill = Purchase price − FV of identifiable net assets
  • Non-controlling interest (NCI) = FV of remaining ownership not held by acquirer
  • Post-acquisition: line-by-line consolidation; intercompany transactions eliminated
FS2

Employee Compensation: Share-Based & Pension

▶

Share-Based Compensation

RSU / Stock Option Expense
Annual SBC expense = Grant-date fair value / Vesting period (years)
Example: 25,000 RSUs × $10 FV = $250,000 total → $83,333/year over 3-year vest
BS offset: Additional paid-in capital ↑ (equity account, NOT liability)
Fair value is locked at GRANT DATE — never remeasured for equity-settled awards
SBC has NO cash outflow from operations. It's added back in operating cash flows (indirect method). It DOES dilute EPS by increasing shares outstanding at vest/exercise.
Treasury Stock Method (Diluted EPS)
Net new shares = Options outstanding × (Market price − Exercise price) / Market price
Add net new shares to EPS denominator
Only dilutive if market price > exercise price (options are in the money)

Defined Benefit Pension

Funded Status & Pension Cost
Funded status = Fair value of plan assets − PBO (Projected Benefit Obligation)
Positive = overfunded | Negative = underfunded → liability on balance sheet

Periodic pension cost = Current service cost + Past service cost + Interest cost on PBO
− Expected return on plan assets + Actuarial gains/losses (amortization)
Under IFRS, expected return on plan assets = discount rate (no assumption). Under US GAAP, companies can set their own expected return assumption → higher assumption → lower reported pension cost → higher earnings. Key analyst adjustment: replace with discount rate to see true economic cost.

Best → Worst: GAAP-compliant + decision-useful → GAAP-compliant but biased (aggressive/conservative) → Non-GAAP or fictitious.

Reporting quality (how things are disclosed) is a prerequisite for assessing earnings quality (what the underlying economics are).

Accruals ratio = (NOA_end − NOA_begin) / Avg NOA

FSA Integration: DuPont & Holistic Analysis

▶

Extended DuPont

5-Factor DuPont
ROE = Tax burden × Interest burden × EBIT margin × Asset turnover × Leverage
Tax burden = NI/EBT | Interest burden = EBT/EBIT | EBIT margin = EBIT/Rev
Asset turnover = Rev/Avg Assets | Leverage = Avg Assets/Avg Equity
Diagnostic Use
Company A and B both have ROE = 15%.
A: Margin 10%, Turnover 1.5, Leverage 1.0 → driven by operational efficiency
B: Margin 3%, Turnover 1.0, Leverage 5.0 → driven entirely by leverage (fragile!)
Same ROE, completely different sustainability. DuPont reveals WHY.

Cash Flow Quality Indicators

  • High quality: CFO consistently > Net income (accruals converting to cash)
  • Warning: CFO persistently < Net income → aggressive accrual recognition
  • CapEx vs D&A: CapEx < D&A persistently → underinvesting, deteriorating asset base
  • FCF = CFO − CapEx; dividend safety requires sustained positive FCF

Dividends & Share Repurchases

▶

Dividend Forms — Effects at a Glance

TypeEquity EffectShare PriceShares Outstanding
Cash dividendAssets and equity ↓Falls on ex-div dateUnchanged
Stock dividendNo change (RE → paid-in capital)Falls proportionallyIncreases
Stock splitNo change to any accountFalls proportionallyIncreases

Stock dividends and splits are cosmetic. P/E and total market cap are UNCHANGED. EPS falls (more shares, same earnings), but so does price → P/E stays the same.

Dividend Policy Theories

  • MM Irrelevance: In perfect markets, dividend policy is irrelevant — investors can create homemade dividends. Value comes from investment policy, not payout.
  • Bird in the Hand: Investors prefer certain dividends today over uncertain capital gains → high dividends ↑ value.
  • Tax Argument: Capital gains taxed at lower rate → retain earnings and let price appreciate.
  • Signaling: Dividend increases signal management confidence. Cuts signal trouble (even if rational).
  • Agency: Dividends reduce free cash flow → constrain wasteful spending by managers.

Share Repurchases

Buyback Accretion Test
Buyback is EPS accretive when: Earnings yield (E/P) > After-tax cost of financing
i.e., 1/(P/E) > after-tax interest rate on debt used to fund buyback
Dividend Safety
Dividend coverage = EPS / DPS = 1 / payout ratio
FCFE coverage = FCFE / (Dividends + Buybacks)
Sustainable dividend requires FCFE coverage ≥ 1

Cost of Capital: Advanced Topics

▶

WACC

WACC Formula
WACC = w_d·r_d(1−t) + w_p·r_p + w_e·r_e
Weights = TARGET capital structure (not current book or market)
Cost of debt = YTM on existing debt | Cost of equity = CAPM, DDM, or bond yield + premium

Cost of Equity

CAPM & Re-Levering Beta
r_e = r_f + β × ERP

Unlever comparable: β_U = β_L / [1 + (1−t)(D/E)]
Re-lever to your D/E: β_L = β_U × [1 + (1−t)(D/E)]
Re-Levering Beta Example
Comparable: β_L = 1.2, D/E = 0.5, t = 30%
β_U = 1.2 / [1 + (0.7)(0.5)] = 1.2 / 1.35 = 0.889
Your target D/E = 1.0:
β_L = 0.889 × [1 + (0.7)(1.0)] = 0.889 × 1.7 = 1.51
More leverage → higher equity beta → higher required return on equity.

EQ1/2

Dividend Discount Models

▶

Gordon Growth Model

GGM & Justified Multiples
V₀ = D₁/(r−g) = D₀(1+g)/(r−g)

Justified P/E = (1−b)/(r−g) b = retention ratio
Justified P/B = (ROE−g)/(r−g)
Justified P/S = Net profit margin × (1−b)/(r−g)
Justified P/B Intuition
ROE=15%, r=12%, g=5%: Justified P/B = (0.15−0.05)/(0.12−0.05) = 1.43×
Trades at 1.43× book because ROE > r → company creates value above cost of equity.
If ROE = r → P/B = 1 exactly (stock = book value)
If ROE < r → P/B < 1 (stock trades below book)

H-Model

H-Model (growth declines linearly from g_S to g_L)
V₀ = D₀[(1+g_L) + H(g_S − g_L)] / (r − g_L)
H = half-life of above-normal growth (e.g., 10-year phase → H = 5)
g_S = initial high growth | g_L = long-run sustainable growth
EQ3

Free Cash Flow Valuation

▶

FCFF & FCFE

FCF Formulas
FCFF = NI + D&A + Interest(1−t) − ΔWC − CapEx
= CFO + Interest(1−t) − CapEx

FCFE = FCFF − Interest(1−t) + Net new borrowing
= CFO − CapEx + Net new borrowing

FCFF → discount at WACC → firm value → subtract debt → equity value
FCFE → discount at r_e → equity value directly
The most common error: when starting from NI to FCFF, forgetting to add back Interest(1−t). FCFF is pre-debt (available to all capital providers), so you must add it back. FCFE is post-debt, so subtract it.

EQ5

Residual Income Valuation

▶

Residual Income Model

RI Formula
RI_t = EPS_t − r × B_{t-1} = (ROE_t − r) × B_{t-1}
V₀ = B₀ + Σ[RI_t / (1+r)^t]
Single-stage: V₀ = B₀ + [(ROE−r)/(r−g)] × B₀
Worked Example
B₀ = $20, ROE = 18%, r = 12%, g = 5%
V₀ = $20 + [(0.18−0.12)/(0.12−0.05)] × $20 = $20 + (0.06/0.07) × $20
= $20 + $17.14 = $37.14
The $17.14 premium exists because ROE > r — the company earns above its cost of equity in every period.

FI2

Arbitrage-Free Valuation

▶

Spreads

Z-Spread vs OAS
Z-spread: constant spread over all spot rates to reprice bond (includes option effect)
OAS = Z-spread − Option cost (strips out option distortion → true credit/liquidity spread)

Callable bond: OAS < Z-spread (call hurts investor)
Putable bond: OAS > Z-spread (put benefits investor)
OAS is the "clean" spread for comparison. Higher OAS vs peers → bond looks cheap relative to model. Lower OAS → expensive. Always use OAS when comparing bonds with different embedded options.

Callable vs Putable — The Comparison Table

Callable BondPutable Bond
ValueV_straight − V_call_optionV_straight + V_put_option
OAS vs Z-spreadOAS < Z-spreadOAS > Z-spread
Effective DurationLower than straight (call caps upside)Lower than straight (put floors downside)
Convexity (low rates)NEGATIVE (price compression)More positive than straight
When rates fallPrice rises LESS (call limits gain)Price rises MORE (put provides floor)
Negative Convexity Intuition
Normal bonds: as rates fall → price rises faster and faster (positive convexity → accelerating gains).
Callable bond: when rates fall below the coupon, issuer is increasingly likely to call the bond.
Investors know this and stop paying more than the call price — the price gets "compressed."
Result: price-yield curve bends BACKWARD = negative convexity. You lose the upside exactly when rates are falling.

FI5

Credit Default Swaps

▶

CDS Mechanics

CDS Structure
Protection buyer: pays fixed CDS spread; receives par on credit event
Protection seller: receives spread; must pay on credit event

CDS spread ≈ credit spread on equivalent bond (no-arbitrage)
Upfront payment = (CDS spread − Standard coupon) × Duration × Notional
Spread > coupon → buyer pays upfront | Spread < coupon → seller pays upfront

CDS buyer profits when spreads WIDEN (credit deteriorates)
CDS Trade Example
You think Company X credit will deteriorate. You BUY CDS protection at 200bps.
Six months later, X has problems → CDS spread widens to 500bps.
You can now SELL offsetting CDS at 500bps → lock in 300bps profit on the spread differential.
This is equivalent to shorting a corporate bond without actually short-selling it.

DV1

Forward Commitments: Forwards, FRAs, Swaps

▶

Forward Pricing

No-Arbitrage Forward Price
No cash flows: F₀(T) = S₀(1+r)ᵀ or S₀·e^(rᶜT)
With carry: F₀(T) = S₀(1+r)ᵀ + FV(costs θ) − FV(benefits γ)
θ = storage costs (add → higher F) | γ = dividends/coupons (subtract → lower F)
Continuous: F₀(T) = S₀·e^((rᶜ+θ−γ)T)
Value of Existing Long Forward at Time t
Vₜ = PV[Fₜ(T) − F₀(T)] = Sₜ − PV[F₀(T)]
Short forward value = negative of the above
Forward Valuation Example
Entered 1-year forward at F₀ = $100. Now t=0.75 (3M left), Sₜ = $105, r = 5%.
New forward price: Fₜ = 105(1.05)^0.25 = $106.29
Vₜ = (106.29 − 100) / (1.05)^0.25 = $6.21
Check: Vₜ = 105 − 100/(1.05)^0.25 = 105 − 98.79 = $6.21 ✓

FRA Pricing (3-Step)

FRA Fixed Rate
Step 1: Timeline (n₁ = h days, n₂ = m days)
Step 2: De-annualize: [1 + r₂(n₂/360)] / [1 + r₁(n₁/360)] − 1
Step 3: Annualize × 360/(n₂−n₁)
FRA 3×9 Example
3M LIBOR = 5.60%, 9M LIBOR = 5.92% (n₁=90, n₂=270)
De-ann = [1 + 0.0592(270/360)] / [1 + 0.056(90/360)] − 1 = 1.04440/1.01400 − 1 = 0.0300
FRA rate = 0.0300 × 360/(270−90) = 0.0300 × 2 = 6.00%

Swap Pricing & Valuation

Fixed Swap Rate
r_FIX = (1 − d_n) / (d₁ + d₂ + ... + d_n)
d_i = 1 / [1 + (LIBOR_i × days_i/360)]
Swap Value Mid-Life (receive-fixed party)
V = NA × (FS₀ − FSₜ) × Σdᵢ
Positive if FSₜ fell below FS₀ (current rates lower → receive-fixed wins)
Swap Valuation Example
7-year swap, receive-fixed at FS₀ = 3%. Now 5 years left, current 5Y rate FSₜ = 1.59%.
Sum of 5Y discount factors = 4.8161, NA = €100M
V = €100M × (0.03 − 0.0159) × 4.8161 = €6.79M gain to receive-fixed party
Rates fell → being locked into 3% is valuable when market is 1.59%.
DV2

Contingent Claims: Binomial, BSM, Greeks, Swaptions

▶

Binomial Model

Risk-Neutral Probability & Option Value
π = (1+r−d)/(u−d) ← risk-neutral probability of up move
c = PV[π·c⁺ + (1−π)·c⁻] where c⁺ = max(0, uS−X), c⁻ = max(0, dS−X)
p = PV[π·p⁺ + (1−π)·p⁻] where p⁺ = max(0, X−uS), p⁻ = max(0, X−dS)
One-Period Call Example
S=€100, X=€100, r=5.15%, u=1.35, d=0.74
S⁺=135, c⁺=35; S⁻=74, c⁻=0
π = (1.0515−0.74)/(1.35−0.74) = 0.311/0.61 = 0.51
c = [0.51(35) + 0.49(0)] / 1.0515 = 17.85/1.0515 = €16.98
American options: at EACH node compare exercise value vs hold value — take MAX. European: never early exercise (exception: American calls on dividend-paying stocks near ex-date may be worth exercising).

BSM Model

BSM Formulas (with dividends γ)
c = S·e^(−γT)·N(d₁) − e^(−rT)·X·N(d₂)
p = e^(−rT)·X·N(−d₂) − S·e^(−γT)·N(−d₁)

d₁ = [ln(S/X) + (r−γ+σ²/2)T] / (σ√T)
d₂ = d₁ − σ√T

γ = δ (dividend yield) for stocks | γ = r_f (foreign rate) for currency options
Memory: Call = S minus PV(X). Put = PV(X) minus S. Calls use N(d₁), N(d₂). Puts use N(−d₁), N(−d₂). N(d₂) = probability call is ITM. N(d₁) = call delta.

Put-Call Parity

Put-Call Parity
c + PV(X) = p + S (no dividends)
c + PV(X) = p + S·e^(−δT) (with dividends)
Rearranged: p = c + PV(X) − S → synthetic put = long call + lend + short stock

Option Greeks

GreekMeaningLong CallLong PutKey Note
Delta (Δ)∂V/∂S — price sensitivity0 to +1−1 to 0= N(d₁) for calls; highest ATM
Gamma (Γ)∂Δ/∂S — delta curvature>0>0Highest ATM near expiry; always positive for long options
Theta (Θ)∂V/∂t — time decay<0Usually <0Opposite sign to Gamma for long options
Vega (ν)∂V/∂σ — vol sensitivity>0>0Long options always benefit from higher vol
Rho (ρ)∂V/∂r — rate sensitivity>0<0Calls: higher r → ↑ value. Puts: higher r → ↓ value
Gamma and Theta are on opposite sides for long options: you benefit from large moves (gamma) but bleed value daily (theta). Short options collect theta but suffer from gamma when markets move sharply.

Swaptions

Payer vs Receiver Swaption
Payer swaption: right to PAY fixed, receive floating. Profitable if market rate rises above R_X.
PAYSWN = (AP) × PVA × [R_FIX·N(d₁) − R_X·N(d₂)]

Receiver swaption: right to RECEIVE fixed, pay floating. Profitable if market rate falls below R_X.
RECSWN = (AP) × PVA × [R_X·N(−d₂) − R_FIX·N(−d₁)]

AP = accrual period | PVA = PV annuity factor | R_FIX = market swap rate at expiry

Futures Return Components & Contango/Backwardation

Total Futures Return
Total return = Price return + Roll return + Collateral return

Backwardation (e.g., oil market during supply crunch):
Oct futures = $80, Dec futures = $75. Roll: sell Oct at $80, buy Dec at $75 → $5 gain per barrel.
High convenience yield = traders pay premium for immediate physical delivery → near-term futures > far = backwardation.

Contango (e.g., oil with abundant supply):
Oct = $75, Dec = $80. Roll: sell at $75, buy at $80 → $5 loss per barrel.

Year 3: rises to $130 → incentive fee only on $130−$120=$10, NOT on $130−$105=$25.
Investor protected from paying twice for recouping lost ground.

PM2

Active Portfolio Management

▶

Key Metrics

Active Return, Active Weights, Information Ratio
Active return R_A = R_P − R_B
Active weight Δwᵢ = w_P,i − w_B,i (must sum to 0)
R_A = Σ(Δwᵢ × RAᵢ)

Information Ratio IR = E[R_A] / σ(R_A) ← reward-to-active-risk
IR ≥ 0.5: good | IR ≥ 1.0: exceptional

Grinold-Kahn: S_P² = S_B² + IR² (portfolio SR depends on benchmark SR + manager skill)

Fundamental Law of Active Management

IR = IC × √BR (basic) | IR = TC × IC × √BR (full)
IC = Information Coefficient = correlation(forecast, actual active returns) = SKILL
BR = Breadth = number of INDEPENDENT investment decisions per year
TC = Transfer Coefficient = correlation between forecasts and actual active weights (portfolio construction constraint)
Why Breadth Matters
Manager A: IC=0.10 (modest skill), BR=100 decisions/year → IR = 0.10 × √100 = 1.0
Manager B: IC=0.20 (good skill), BR=25 decisions/year → IR = 0.20 × √25 = 1.0
Same IR despite different skill levels — A compensates with more bets.
Key insight: diversifying your alpha sources (more independent bets) is as valuable as being smarter.

ETFs: Mechanics & Applications

▶

Creation/Redemption Arbitrage

  • ETF premium (price > NAV): AP buys basket of securities, delivers to issuer, receives ETF shares, sells ETF → drives price back to NAV
  • ETF discount (price < NAV): AP buys ETF shares, redeems for basket, sells securities → drives price back to NAV
  • This mechanism keeps ETF prices very close to NAV continuously

Tracking Error Sources

  • Management fees and transaction costs
  • Index rebalancing and reconstitution costs
  • Securities lending income (reduces tracking error)
  • Cash drag (ETF holds cash, index doesn't)

PM5

Measuring & Managing Market Risk: VaR

▶

VaR Definition

VaR: Minimum Loss at Given Confidence
"5% daily VaR = $2M" → on 5% of days, loss WILL BE AT LEAST $2M

Normal distribution z-scores:
1% VaR → 2.33σ below mean
5% VaR → 1.65σ below mean
16% VaR → 1.00σ below mean
Parametric VaR
Daily VaR (5%) = μ_daily − 1.65 × σ_daily (as loss: 1.65 × σ − μ if μ ≈ 0)
Scale: σ_T-day = σ_1-day × √T → VaR_monthly ≈ VaR_daily × √21

VaR Methods Compared

MethodKey AssumptionProCon
ParametricNormal distributionSimple, closed-form, fastFails for non-normal tails; bad for options
Historical SimulationPast distributions repeatCaptures fat tails; no distribution assumedLimited by history; can miss new regimes
Monte CarloSpecified factor distributionsFlexible; handles complex instrumentsComputationally intensive; model risk
VaR only tells you the MINIMUM loss at the threshold — it says nothing about how BAD losses are beyond that. CVaR (Expected Shortfall) = expected loss GIVEN you're in the tail. CVaR is more informative than VaR for tail risk.

Mosaic Theory — The Classic L2 Scenario

Worked Example
Analyst reviews: (1) SEC filing showing rising DSO [public], (2) industry report on slowing demand [public], (3) public earnings call where CFO mentions "channel headwinds" [public], (4) calls a distributor who confirms slower orders [public conversation].

Is this MNPI? No. Each piece was public. The conclusion is nonpublic but built from public information → Mosaic Theory = legal and ethical.

What WOULD be MNPI? If the CFO privately said: "We're going to miss by 30%" before the public announcement.

GIPS Key Rules

  • Voluntary, firm-wide standards for presenting investment performance
  • All actual, fee-paying, discretionary portfolios must be in at least one composite (no cherry-picking)
  • Must present 5 years of GIPS history (or since inception if less), then build to 10
